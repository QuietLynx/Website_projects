Assets folder

Put your images and media here. Suggested structure:

- project1.jpg (used in the first project card)
- project1-poster.jpg (poster for videos, optional)
- project2.jpg
- logos/ (stack icons, optional)

Tips
- Prefer WebP or compressed JPG; aim for ≤300 KB per image where possible.
- Use ~16:9 (e.g., 1200×675) for project hero thumbnails to match the layout.
- Name files clearly: `project-name-version.jpg`.
- If you add videos, prefer WebM/MP4 and include a poster image.
